# 📊 Matplotlib 数据可视化展示平台

## 🚀 快速开始

### 部署到 Cloudflare Workers
```bash
wrangler deploy
```

### 访问地址
```
https://diliuzhang.3239251138.workers.dev
```

## 📋 功能特色

### 🎯 6个交互式图表
1. **🌪️ 深圳市24小时平均风速**
   - 支持折线图/柱状图切换
   - 显示平均风速统计信息

2. **📈 正弦与余弦函数曲线**
   - 数学函数可视化
   - 支持单独显示/隐藏

3. **📊 多坐标轴数据对比**
   - 双Y轴设计
   - 柱状图+折线图组合

4. **⚙️ 自定义刻度样式**
   - 刻度旋转和格式化
   - 自定义刻度标签

5. **🎨 轴脊隐藏效果**
   - 简洁设计风格
   - 可切换网格显示

6. **🎯 移动轴脊位置**
   - 中心坐标系
   - 数学坐标系风格

## 🎮 交互功能

### 筛选功能
- 全部图表
- 风速数据
- 数学函数
- 坐标轴

### 键盘快捷键
- `Ctrl + R`：重置所有图表
- `Ctrl + A`：播放所有动画

### 图表控制
- 切换图表类型
- 播放动画效果
- 重置图表状态
- 随机数据生成

## 📱 响应式设计
- 桌面端：2列网格布局
- 移动端：单列布局
- 触摸友好的交互设计

## 🛠️ 技术栈
- **Chart.js**: 图表渲染
- **Cloudflare Workers**: 部署平台
- **CDN**: 资源加速
